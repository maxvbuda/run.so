# Run.so

A runbook automation platform for managing and executing workflows.

## Features

- **Runbooks** - Create and manage automated runbooks
- **Dashboard** - Overview of your automation workflows
- **Status** - Monitor execution status
- **Runs** - Track and review past executions
- **Integrations** - Connect with external services (Browserbase, etc.)

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/maxvbuda/run.so.git
cd run.so
```

2. Open `index.html` in your browser, or serve locally:
```bash
npx serve .
```

## Usage

1. Create a new runbook from the Runbooks page
2. Write your automation instructions in the editor
3. Add integrations as needed
4. Publish and run your runbook
